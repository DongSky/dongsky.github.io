  • Learnable Bounding Box Adjusters

    • Project Page | PyTorch Implementation

    We aim to build a transfer learning-based weakly-supervised object detection (WSOD) method, to stably improve the localization performance of current WSOD framework, thus imporving the overall detection accuracy. This bounding box adjuster can be easily adopted into various WSOD training frameworks.

  • Few-shot Vision Transformer

    • Project Page | PyTorch Implementation

    The project starts with the observation that naïve application of ViT to few-shot learning results in substantial performance degradation. To resolve this issue, this project introduces a new few-shot training framework for ViTs, named self-promoted supervision (SUN). This framework aims to enhance the capability of learning the token dependencies via location-specific supervision. As a result, SUN reports high performance on few-shot learning benchmarks. It can be considered as a good baseline for ViTs on few-shot learning.
